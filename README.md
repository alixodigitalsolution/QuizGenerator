# QuizGenius AI 🧠

An advanced, AI-powered assessment generator that transforms your study materials, notes, or topics into professional quizzes in seconds.

![Project Preview](https://via.placeholder.com/800x400.png?text=QuizGenius+AI+Interface)

## 🌟 Features

- **Multi-Source Generation**: Upload PDFs, paste text, or simply provide a topic prompt.
- **AI-Powered Engine**: Leverages Google Gemini Flash for intelligent question crafting and professional explanations.
- **Customizable Assessments**: Choose difficulty levels (Beginner, Intermediate, Advanced) and question types.
- **Interactive Quiz UI**: Real-time feedback, timers, and progress tracking.
- **Premium Design**: Modern, responsive glassmorphism interface built for excellence.

## 🚀 Quick Start

### Prerequisites

- Python 3.8+
- Google Gemini API Key

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/alixodigitalsolution/QuizGenerator.git
   cd QuizGenerator
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Configure Environment:**
   Create a `.env` file in the root directory:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

4. **Run the Application:**
   ```bash
   python app.py
   ```
   The app will be available at `http://127.0.0.1:5000`.

## 🛠️ Tech Stack

- **Backend**: Flask (Python)
- **AI**: Google Generative AI (Gemini)
- **Frontend**: Vanilla HTML5, CSS3 (Glassmorphism), JavaScript
- **Styling**: Modern CSS Grid & Flexbox with Inter Typography
